Deliberation RuleML 1.02 XSLT Compactifier Directory

This directory contains several XSLT transformations (1.02_compactifier.xslt, 1.02_compactifier-ifthen.xslt), defined modularly (1.02_deliberation_compactifier_module.xslt, 1.02_deliberation_compactifier-ifthen_module.xslt) applicable to documents in the Deliberation RuleML 1.02 language, that is applicable to all documents in the Deliberation RuleML 1.02 language, producing documents in the compact serialization.

See the XSLT-Based_Normalizer section of the Deliberation RuleML 1.02 Specification for documentation of the transformation..

See the XSLT Usage Appendix of the Deliberation RuleML 1.02 Specification for hints on how to apply the Upgrader XSLT to RuleML documents.
