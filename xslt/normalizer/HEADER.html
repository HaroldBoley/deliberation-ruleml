Deliberation RuleML 1.02 XSLT Normalizer Directory

This directory contains an XSLT transformation (1.02_normalizer.xslt), defined modularly (1.02_normalizer_module.xslt and 1.02_pretty-print_module.xslt), that is applicable to all documents in the Deliberation RuleML 1.02 language, producing documents in the normalized serialization.

See the XSLT-Based_Normalizer section of the Deliberation RuleML 1.02 Specification for documentation of the transformation..

See the XSLT Usage Appendix of the Deliberation RuleML 1.02 Specification for hints on how to apply the Upgrader XSLT to RuleML documents.
