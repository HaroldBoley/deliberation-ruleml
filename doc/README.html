This directory contains automatically-generated schema docs for several anchor languages of the Deliberation RuleML 1.02 family. The normalized serialization is documented, for brevity. A selection of anchor languages is made available, ranging from the supremum language nafholog to the minimal datalogplus. The naffologeeq language, which was the supremum language of previous version from X.Y to 1.02, is also available for comparison purposes. Additionally, an intermediate language, disdatalogplus_mid demonstrates how MYNG may be used to define precisely a customized language including (only) selected features. Please see the schema docs section of the Deliberation RuleML 1.02 Content Models document for details on the generation process of the schema docs from the Relax NG pivot schemas
